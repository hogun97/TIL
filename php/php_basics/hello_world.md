# Hello, World!

```php
<html>
<head></head>
<body>
Hello, World!
</body>
<?php ?>
</body>
</html> 
``` 
Above is the starting point in PHP - a simple HTML page.

Example:

```php
<?php $user = "John"; ?>
<html>
<head></head>
<body>
Hello <?php echo $user; ?>!
</body>
</html>
```

Above will print: Hello John!
